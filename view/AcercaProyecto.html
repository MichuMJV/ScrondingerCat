Scröninger’s box

Proyecto Semestral Programación 2 (Prototipo y Diseño Web)

Profesor: Juan Antonio Zamora

Objetivo: Poner en práctica los conceptos aprendidos en clases con la confección de un prototipo y diseño web funcional ESTÁTICO de un proyecto de desarrollo de sitio web utilizando las herramientas de HTML para la estructura del sitio y CSS para el diseño y estilos.

este sitio está creado especificamente para ser una plataforma para subir spoilers de peliculas y series, cuando entras no sabes con lo que te encontrarás, por lo que es ambigüo si te vas o no a spoilear, de ahí su nombre, Scröninger’s box

el logo, la paleta de colores, la tipografía y la estructura fueron elegidos el primer día, el logo ha sido creado a partir de dos modelos 3D de un gato de uso libre y una caja cualquiera

el contraste, el feedback, la iconografía y el etiquetado de botones y funcionalidades ha sido cuidado al mas minimo detalle.
